# Глосарій (Ubiquitous Language)

* **Task (Завдання)**: Основна сутність системи. Елемент списку, який має бути виконаний. Має назву та статус.
* **Status (Статус)**: Стан, у якому перебуває Завдання. Наприклад: "pending" (в очікуванні), "completed" (виконано).
* **List (Список завдань)**: Контейнер, що містить усі Завдання.
